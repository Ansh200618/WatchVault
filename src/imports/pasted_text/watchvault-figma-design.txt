Design a premium native Android app UI in Figma for an application called “WatchVault”.

App type:
Native Android APK app, not a website.

App purpose:
WatchVault is a premium movie, series, anime, and episode tracking app where users can discover content, view details, track watch progress, mark episodes watched, manage upcoming releases, add reminders, and view personal stats.

Design style:
Use Material Design 3 / Material You principles with a premium SaaS-style entertainment layout.

Overall look:
- Clean premium Android UI
- Soft off-white background
- Large rounded cards
- Black primary buttons
- Smooth pill filters
- Floating content panels
- Cinematic poster-based layout
- Minimal luxury feel
- Clean spacing
- Rounded bottom navigation
- Modern app-store-quality interface
- Native Android mobile layout only
- No website-style layout

Theme:
Create both Light and Dark versions.

Default Light Theme:
- Background: #F8F6F1
- Surface/Card: #FFFFFF
- Primary text: #111111
- Secondary text: #6E6E6E
- Primary button: #111111
- Button text: #FFFFFF
- Accent: #1DBFA7 or #14A890
- Soft border: #E8E3D8
- Warning/upcoming accent: #FFB84D

Dark / AMOLED Theme:
- Background: #000000
- Surface/Card: #101010
- Primary text: #FFFFFF
- Secondary text: #A5A5A5
- Primary button: #FFFFFF
- Button text: #000000
- Accent: #1DBFA7
- Border: #242424

Typography:
Use a modern sans-serif font like Inter, SF Pro, or Google Sans style.

Text sizes:
- Large screen title: 28sp bold
- Section heading: 20sp semi-bold
- Card title: 16sp semi-bold
- Body text: 14sp regular
- Metadata text: 12sp medium
- Button text: 14sp semi-bold
- Bottom nav label: 11sp medium

Corner radius:
- Main cards: 28dp
- Poster cards: 24dp
- Buttons: 50dp pill
- Filter chips: 50dp pill
- Bottom navigation: 32dp
- Small icon buttons: circular
